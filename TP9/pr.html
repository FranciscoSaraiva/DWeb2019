ABSTRACT:

Trabalho de casa 9 era preciso construir uma aplicação de linha de comandos para fazer a importação de ficheiros json para uma base de dados mongo.

Foram usadas as seguintes tecnologias para suporte na criação desta aplicação de linha de comandos: boxen para os efeitos de caixa na linha de comandos, chalk para pintar com cores palavras nas linhas de comandos, node-cmd para apoio a correr comandos, prompt-confirm para rápidas mensagens de confirmação por parte do utilizador e yargs para apoio no desenvolvimento de aplicações de linhas de comando.

Deliverables:

  1. Instruções da aplicação
  2. Ficheiro de aplicação
